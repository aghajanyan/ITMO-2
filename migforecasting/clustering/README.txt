Clust-data (k-means).py - главный модуль кластеризации данных и их всестороннего анализа

Различные эксель-файлы являются выходными результатами анализа данных в модуле Clust-data (k-means).py 

Clust-error (k-means).py - циклический запуск алгоритма кластеризации для оценки субоптимального количества кластеров

recommendation system - директория с модулем, реализующим рекомендации по изменению социоэкономических индикаторов для произвольного поселения

datasets - директория с датасетами для кластеризации
