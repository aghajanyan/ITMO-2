agestruct (per age) - директория с файлами половозрастной структуры от 0 до 69, где каждый столбец [0, 1, 2, ..., 69] отражает возраст, а строка конкретное мун. образование согласно полу (отдельные строки male и female)

agestruct (per cohort) - директория с файлами половозрастной структуры в когортном представлении, где каждый столбец ['0-4', '5-9', ..., '65-69'] отражает когорту -- || -- ||

agestruct (prop) - вместо количества содержит данные о долях в каждой когорте от общего числа людей (для взвешенного сравнения половозрастных структур между мун. образованиями)

popdataprep.py - предобработка данных "Численность всего населения по полу и возрасту на 1 января текущего года" из БД ПМО (получено с сайте tochno.st, файл 8112014)

popconnecter.py - соединяет и сортирует данные по мужчинам и женщинам, после отработки popdataprep.py

cohortmaker.py - осуществляет преобразование данных за каждый год в когортное представление

propmaker.py - осуществляет преобразование абсолютных значений в когорте в долевое представление
