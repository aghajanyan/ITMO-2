superdataset (full data).csv - полный датасет со всеми отобранными признаками (без нормализации!)

superset features.xlsx - данные об отобранных признаках датасета

features separately - каждый отобранный признак в отдельном csv файле (название признака = название файла)

mig data - данные прибывших и выбывших в отдельных файлах по годам. На основе разницы получено сальдо.

training ready - готовые для обучения датасеты.

PrepFullSuperDataset.py - модуль сборки полного датасета на основе соединения с признаком миграционного сальдо

getsaldo.py, rawdataprep.py - обработчики данных
