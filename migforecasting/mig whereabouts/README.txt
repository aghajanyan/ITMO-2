inflow LO.csv & outlfow LO.csv - файлы с данными о прибвыших и выбывших по Ленобласти

saldo LO.csv - файл с миграционным сальдо по Ленобласти

getsaldo.py - модуль вычисления сальдо (разница прибывших и выбывших)

getprop.py - вычисление средних долей распределения миграционного потока по соответствующему направлению перемещения

migprop (avg 3years).csv - файл с долями распределения миграционных потоков по направлению перемещения (средняя за последние 3 года)

migprop (avgalltime).csv - файл с долями распределения миграционных потоков по направлению перемещения (средняя за всё время)

inflow employable LO.csv & outflow employable LO.csv - файлы с данными о прибвыших и выбывших ТРУДОСПОСОБНЫХ граждан по Ленобласти

migprop employable (avg alltime).csv - файл с долями распределения миграционных потоков трудоспособных по направлению перемещения от числа трудоспособных (средняя за всё время)

migprop empl from total (avg alltime).csv - файл с долями распределения миграционных потоков трудоспособных по направлению перемещения от ОБЩЕГО числа миграции (средняя за всё время)
